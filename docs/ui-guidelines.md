# UI Guidelines

## General Style

The UI should feel:
- modern
- clean
- minimal
- intuitive

---

## Design Priorities

Prioritize:
- readability
- spacing consistency
- responsive layouts
- simple navigation

---

## Components

Prefer reusable components for:
- cards
- forms
- tables
- dialogs
- filters
- dashboards

---

## Layouts

Use:
- responsive grids
- clean spacing
- clear visual hierarchy

Avoid cluttered screens.

---

## Colors and Themes

Support:
- dark mode
- light mode

Ensure:
- proper contrast
- consistent colors
- readable typography

---

## Forms

Forms should:
- be simple
- minimize user effort
- provide clear validation feedback

---

## Tables and Lists

Lists should support:
- filtering
- sorting
- pagination when necessary

---

## Statistics and Charts

Use MUI X Charts for:
- sales statistics
- order statistics
- popularity metrics
- performance dashboards

Charts should remain:
- simple
- readable
- visually consistent

---

## Mobile Responsiveness

Even though this repository is web-only:
- interfaces must remain usable on smaller screens.

Avoid layouts that break on mobile widths.
